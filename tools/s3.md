# S3

## Поднять S3 локально

minio

```yaml
version: '3.5'

services:
  s3:
    image: minio/minio:latest
    entrypoint: bash -c 'mc mb -p data/data-testing && /usr/bin/docker-entrypoint.sh minio server /data'
    environment:
      - MINIO_ROOT_USER=admin
      - MINIO_ROOT_PASSWORD=password
    ports:
      - 9000:9000
    volumes:
      - s3_buckets:/data
    restart: unless-stopped

volumes:
  s3_buckets:

```

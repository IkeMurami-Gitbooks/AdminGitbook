Какие домены первого уровня для чего

```
com — commerce: открыл компанию — заведи домен  mycompany.com
app — application: например написал приложение, заведи домен myapp.app
org — non-commerce — для некоммерческих организаций домен
pw — ну вы поняли
io — вообще это домен какого-то региона, но пошла мода opensource там поднимать
```
